# FastAPI Calculator with History & Report

This project is a small calculator web application built with FastAPI and SQLAlchemy.  
It features:
- Basic operations (+, -, *, /, ^)
- Calculation history
- Report with total calculations, sum of results, average result, and counts per operation
- Unit, integration, and E2E tests
- Docker image and CI pipeline (GitHub Actions) that runs tests and pushes to Docker Hub

## Requirements

- Python 3.10+
- Node.js 18+ (for Playwright)
- Docker (for containerization)

## Setup

pip install -r requirements.txt
npm install
npx playwright install
